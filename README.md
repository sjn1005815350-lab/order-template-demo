# 报单排序模板 Demo

这是一个纯静态前端 Demo，用于演示三段式报单排序配置页面。

## 使用方式

直接用浏览器打开：

- `index.html`：报单排序模板配置
- `PRD.html`：网页版 PRD，可直接复制表格
- `PRD.md`：Markdown 版 PRD

如果希望通过本地服务访问，可以在当前目录执行：

```bash
python -m http.server 5173
```

然后访问：

```text
http://localhost:5173
```

## 已包含功能

- 顺序配置：维护不同渠道的报单顺序，每个顺序有独立名称，展开后通过拖动块调整对冲渠道排序
- 模板设置：将不同品种配置拼装成完整交易模板，每个品种配置引用顺序名称，交易品种支持下拉多选编辑
- 客户模板应用：客户交易账户绑定最终应用的交易模板，支持按交易账户查询和重置
